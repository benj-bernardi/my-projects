# Mini Projects

Collection of small scripts for practice and learning, primarily using JavaScript.

## Projects Included

- **JavaScript Calculator**: A simple calculator built with JavaScript to perform basic arithmetic operations (addition, subtraction, multiplication, and division).
- **Dice Roller**: Simulates dice rolls for common RPG dice types, including 1d4, 1d6, 1d8, 1d12, 1d20, and 1d100.
- **To-Do List**: A basic task manager that allows users to add, mark, and remove tasks.
- **Guess the Number**: A number guessing game where the player must guess a random number between 1 and 15.

## How to Run

### JavaScript Projects
1. Navigate to the project folder.  
2. Run the script using Node.js:
   ```bash
   node filename.js

## Purpose

These mini projects were developed to improve programming skills and explore fundamental concepts in JavaScript, including logic, user interaction, and data manipulation.
